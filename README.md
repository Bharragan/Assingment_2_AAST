## Description

Implementation of 5 exercises covering signal processing, time series forecasting, IMU analysis, audio separation, and activity classification.

## Quick Start

```bash
pip install -r requirements.txt
python exercise_X/main.py
```

## Exercises

1. **DTFT & DFT** - Signal analysis and windowing effects
2. **AirPassengers** - ARIMA, SARIMA, Holt-Winters forecasting
3. **IMU Tracking** - 3D displacement with ZUPT
4. **FastICA** - Blind audio source separation
5. **Activity Recognition** - SVM classification

## Requirements

- Python 3.8+
- numpy, scipy, pandas
- matplotlib, seaborn
- statsmodels, sklearn
- scikit-learn
